URL / Oque é.

Exemplo = "Https//www.youtube.com/link"

URL "Uniform Resource Locator" É o link completo

Https// =

www = Sub-Dominio principal

Sub-Dominio = pode conter elementos da sua escolha

youtube.com = Dominio

.com = TLD / Pode conter GTLD ou CCTLD

.com = para fins comerciais

.edu = para instituições educacionais

.gov = para instituições governamentais

.store = para lojas

.br , .us , .uk, .ru = são CCTLD's que dizem o pais.

/link = caminho "Aonde o conteudo do seu site está localizado na hospedagem."
